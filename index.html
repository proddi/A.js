A.js

A.js is a minimalistic animation framework for javascript. Of course it can animate dom-nodes but it can also animate properties at your own object. Thats needed for animate visuals on canvas or e.g. on a maps-application where you don't have a fully features rendering engine-api.

.animate(...)

.animate() applies an animation to the object. The function accepts 2 parameters: #1 {} the target values and #2 the options.

Play 


A("#n1")
    .animate({ width: 150 })
    .delay(2000).reset()
    ;

Chaining

It's easy to create a chain by just apply more functions.

Play 


A("#n2")
    .animate({ width: 150 })
    .animate({ height: 150 })
    .animate({ width: 250, height: 50, backgroundColor: "red" })
    .delay(2000).reset()
    ;

Colors

The framework comes with color support.

Play

COLOR

A("#n3")
    .animate({
            backgroundColor: "#225533",
            borderColor: "red",
            color: "white"
        }, { duration: 2500 })
    .delay(2000).reset()
    ;

.set()

.set({}) is applying the values immediately to the element.

Play 


A("#box-set")
    .set({ width: 150, backgroundColor: "blue" })
    .delay(1000)
    .reset()
    ;

.reset()

.reset() sets the value of the object back to their initial values.

Play 


A("#box-reset")
    .set({ width: 150, backgroundColor: "blue" })
    .delay(1000)
    .reset()
    ;

.then()

.then(fn) is triggered when the animation is done

Play 


A("#box-then")
    .animate({ marginLeft: 150 }, { duration: 2000 })
    .then(function() { alert("Done!"); })
    .delay(2000).reset()
    ;

Defaults


A.defaults = {
  duration: 500,
  easing: "in",
};
